Het ab-actiaal prikbord voor intern gebruik. Vereist VGSR.

== Description ==

Create prikbord items for internal access only. This plugin uses is_user_vgsr() to check for user accessability. Requires VGSR plugin.

== Installation ==

1. Download VGSR Prikbord.
2. Upload the 'vgsr-prikbord' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate VGSR Prikbord from your Plugins page.
4. Create a new Prikbord Item in Dashboard > Prikbord.
